# Seismic Activity Prediction using Deep Learning

## Overview
This project predicts seismic activity trends using CNN + LSTM models.

## Features
- Spatial + Temporal modeling
- Multiple model comparison
- 7-day forecasting
- Visualization (heatmaps, maps, 3D plots)

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Matplotlib

python main.py
